HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • You close sections like the head or the body or sections in the body with the element name.
  • title defines the title of the page
  • lang = defines the language i.e. en for English
  • h1 to h6 are the level of heading a given text block represents.
  • p represents a paragraph.
  • ul, ol, and li represent unordered lists, ordered lists, and list items.
  • img element contains information about images that are diplayed on the webpage.
  • src attribute defines the location of the image.
  • br element creates a line of empty space, or a line break, between two blocks of content.
  • a element, anchor, creates links to the same webpage or other webpages
  • hred attribute points to the URL for the link.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout branch-name: moves to a certain branch
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: stages the changes.
  • git commit -m "message": commits these changes to the repository.
  • git pull origin main: makes sure the current branch is up to date before opening a pull request.
  • git push origin branch-name: pushes up all the changes. Then pull the request on GitHub, making sure the base branch is main and the comparison branch is the other brand. Then merge. All the changes should be on the main branch now.
  • To make sure the local main branch is updated now, enter git checkout main in the command line, and then git pull origin main to pull the remote main branch into the local main branch.

JavaScript

  • JavaScript works with our webpages, and you can see how by using Chrome Developer Tools (go to inspect).
  • var variable_name = "expression": Declared a variable and gives it a name to reference information stored in the variable in the code.
  • The expression of the variable, in the quotation marks, can be a list or array.
  • console.log(variable_name) is a method to output a message to the web console (Chrome DevTools, inspect) by adding an argument.
  • Connect css to html by adding a script into the html code after end of footer and before end of body.
  • Use for loops to perform a task on all items in the array in the expression by allowing a single code block to be executed over and over, rather than manually write lines of code for each item in the array, represented by an index number, 0 being the first one.
  • In a for loop, var x = 0 means the starting point for the loop is the first index of the array, alwasy 0.
  • In a for loop, x is less than variable_name.lenth will keep the loop running because it executes one time for each item in the array.
  • In a for loop, x++ means every time the code loops, x will increase by 1
