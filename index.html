HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • You close sections like the head or the body or sections in the body with the element name.
  • title defines the title of the page
  • lang = defines the language i.e. en for English
  • h1 to h6 are the level of heading a given text block represents.
  • p represents a paragraph.
  • ul, ol, and li represent unordered lists, ordered lists, and list items.
  • img element contains information about images that are diplayed on the webpage.
  • src attribute defines the location of the image.
  • br element creates a line of empty space, or a line break, between two blocks of content.
  • a element, anchor, creates links to the same webpage or other webpages
  • hred attribute points to the URL for the link.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
